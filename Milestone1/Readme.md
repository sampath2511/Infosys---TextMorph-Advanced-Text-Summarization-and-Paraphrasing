Milestone 1 establishes the basic setup of the project by creating a Streamlit-based web application environment. It installs the required dependencies, implements JWT-based authentication for secure login, and launches the Streamlit server. The application is exposed to the internet using Ngrok, allowing users to access it through a public URL. This milestone provides the foundation for adding advanced features like text summarization, paraphrasing, and readability analysis in later stages.

Features:

Streamlit web application setup

Automatic installation of required libraries

Secure JWT authentication system

Token generation with expiration for user sessions

Streamlit server initialization and monitoring

Public deployment using Ngrok

Base structure for future NLP features (summarization and paraphrasing)
